# Brand and asset review

This repository reproduces the look of an existing WorldPlay experience for internal concept evaluation. Inclusion in the prototype is not evidence of production usage rights.

| Asset group | Prototype source | Approval needed before external release |
| --- | --- | --- |
| WorldPlay logo | Supplied site reference | WorldPlay brand owner approval |
| Graphik font files | Local prototype assets | Confirm organisation licence permits hosted web use |
| WorldPlay dashboard imagery | Supplied/reference experience | Confirm copyright and campaign usage scope |
| Racing hero artwork | Supplied concept artwork | Confirm creator, licence and sponsor/brand permissions |
| Horse, jockey and race data | Sample prototype data | Replace with approved official data source |

Record the approver, approval date, permitted channels and expiry for every item before moving beyond a private stakeholder preview.
